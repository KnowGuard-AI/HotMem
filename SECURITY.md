# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in HotMem, please report it responsibly.

**Email:** security@knowguard.ai

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will acknowledge receipt within 48 hours and aim to provide a fix or mitigation within 7 days for critical issues.

## Scope

HotMem is a local-first sidecar designed to run on `127.0.0.1`. It does not include authentication or encryption by default. If you expose HotMem to a network, you are responsible for securing access.
